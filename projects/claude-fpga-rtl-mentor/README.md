# claude-fpga-rtl-mentor

Claude-mediated self-study for the FPGA architect transition. Covers
structured reading of:

- Kilts — *Advanced FPGA Design*
- Maaref — *Advanced Digital System Design Using SoC FPGAs*
- Arora — *FPGA-Based Prototyping Methodology Manual*
- Hennessy & Patterson — *Computer Architecture: A Quantitative Approach*
- AMD vendor docs: UG949, Versal trilogy
- Methodology: arc42, ADRs, NASA GSFC systems engineering

## Files

- `state.md` — canonical current state. Uploaded to the Claude Project KB.
  Tracks: current chapter/topic, cumulative learning, active gaps.
- `notes.md` — working scratchpad.
- `checkpoints/` — per-chat artifacts from the `checkpoint` skill.
- `decisions/` — methodological decisions (e.g. "switch reading order",
  "adopt arc42 for SFU").
